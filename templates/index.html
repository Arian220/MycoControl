Welcome to the Mushroom Cultivation Project

This project aims to develop a control and monitoring system for environmental conditions to optimize mushroom cultivation.

Using technologies such as Raspberry Pi and sensors, we seek to create an ideal environment for mushroom growth, facilitating production and ensuring quality.

We hope this system will help improve cultivation practices and contribute to sustainability in agriculture.
